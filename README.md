# Train-Schedule

This is a simple train schedule made for my computer coding class at the UofU.

## Instructions
Type in a train name, a first train time, and a frequency and press submit to add your train to the schedule.

### Technologies used 
* HTML5
* CSS
* Javascript
* Jquery
* Firebase
* API
* moment.js
